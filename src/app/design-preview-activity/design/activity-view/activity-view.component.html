Image Labeling

Students drag labels to individual docks, which may include leader lines that point to specific areas to be identified.

Grouping

Students group labels according to classification or category by dragging all related labels into a single corresponding dock.

One Label to One Dock

Each label in the activity appears once. It can only be dragged to one dock.

Same Label to Multiple Docks

  • Display Label Once

    Each label in the activity appears once, but multiple docks can be created for the same label. (Student should be able to choose the same label as the answer for multiple docks)

  • Display Each Label Instance

    Each label in the activity appears once in authoring, but multiple docks can be created for the same label. (Student should be able to see the same number of docks as instances of the label )

One Label to One Dock

Each label in the activity appears once. It can only be dragged to one dock.

Same Label to Multiple Docks

  • Display Label Once

    Each label in the activity appears once, but it may be used in one or several docks. Each dock, however, can still only accept one unique label.

  • Display Each Label Instance

    Each label in the activity appears once, but it may be used in one or several docks. Each dock, however, can still only accept one unique label. (Student should be able to see each instance of a label whether or not the label is unique or duplicate)
